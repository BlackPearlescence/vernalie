Vernalie import fixtures
========================

These files are fake nursery spreadsheets for testing the CSV/XLSX importer and
column-mapping flow. They intentionally use different header names and stock
mixes so the importer can prove it handles real-world nursery spreadsheets.

Files:
- green-valley-fruit-trees.csv
- riverbend-greenhouse-vegetables.csv
- meadowbrook-herbs-perennials.csv
- northern-rootstock-mixed.xlsx
- suncrest-annuals-plugs.xlsx
